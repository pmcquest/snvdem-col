# Repository for "Subnational democracy in Colombia" (snvdem24)
This is a repository for the project studying subnational democracy in Colombia, using V-Dem data. Started in 2024 as a single-year (2018) cross-sectional pilot, the project has since expanded into a full 2000-2023 municipality-year panel -- hence the two data eras below. In this repository you'll find: 
- data (see [data/README_data.md](data/README_data.md)) for quantitative analysis, including:
  - [data/2018pmq/](data/2018pmq/) -- the original 2018 cross-sectional pilot (ArcMap layers/shapefiles for a single year), superseded by the panel below but kept for reference
  - [data/panel/](data/panel/README_panel.md) -- the current 2000-2023 municipality-year panel pipeline that produces the SNVDEM index
- [scripts](scripts/) for data analysis
- [bibliography](biblio/) on subnational democracy and V-Dem
- [report](report/) in draft form
- [administrative](admin/) files such as data sharing agreements and team meeting notes
- [session logs](data/panel/claude-md/) documenting Claude Code pipeline-revision sessions (handoffs, next-session notes)

# Description of project
-   **Objective**: Create a granular measure of democratic unevenness in one country (Colombia) using V-Dem.
-   **Relevance**: Method for systematizing background concepts included in subnational V-Dem survey questions. Can provide input for answering substantive questions, such as whether development outcomes in subnational areas are correlated with free and fair elections, or with pressure from a mobilized civil society (civil liberties).
-   **Measurement**: Unevenness is measured using V-Dem data on subnational elections and respect for civil liberties [2 surveys: (sub-national) Elections and Civil Liberties].
-   **Scope**: Originally a 2018 cross-section (`data/2018pmq/`); expanded to a full 2000-2023 municipality-year panel (`data/panel/`) to support trend analysis over time, not just a single-year snapshot.
